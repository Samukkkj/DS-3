TESTE DE SOFTWARE

Teste de Componentes - ocorre quando todos os componentes são testados isoladamente;

Teste de Montagem - componentes são testados em conjuntos;

Teste de Produto - software é testado para confirmar os requisitos funcionais do projeto;

Teste de Integridade do Sistema - ocorre quando o sistema é testado com outros sistemas para a observância dos requisitos de integração;

Teste de Aceitação do Utilizador - testes e utilizadores finais para que atenda suas expectativas.
